## English
App Management

App Management is an application that helps you manage applications and web browsers on your computer.

<br>

![รูปภาพ](https://api-img.purity64.com/AppManagement/img/1.png)
<br>

The application is divided into the following features:

# Browser

This feature allows you to configure which websites will open together with a selected browser.

For example, you can define websites 1, 2, and 3, choose Chrome as the browser, and select Profile 1.
When launched, Chrome will open using Profile 1 and automatically load websites 1, 2, and 3.

<br>

You can also configure additional programs to be launched alongside the browser.

<br>

![รูปภาพ](https://api-img.purity64.com/AppManagement/img/2.png)

Adding Other Categories

This feature allows you to create custom categories for organizing applications.

<br>

![รูปภาพ](https://api-img.purity64.com/AppManagement/img/4.png)

![รูปภาพ](https://api-img.purity64.com/AppManagement/img/3.png)
<br>

Within a created category, you can add applications that you want to launch, such as Visual Studio Code.

<br>


![รูปภาพ](https://api-img.purity64.com/AppManagement/img/5.png)

<br>

# Settings

Background Settings

You can choose either videos or images as the background.

<br>
